how to install

 **use npm**
 -- install package
 ```bash
 npm install
```

-- run app
```bash
npm start
```

-- run build
```bash
npm run build
```

**use yarn** 

-- install package
```bash
yarn install
```

-- run app development
```bash
yarn start
```

-- run build 

```bash
yarn run build
```
